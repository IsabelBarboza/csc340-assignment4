# csc340-assignment4

This project is a REST API built with Spring Boot that manages characters from the Dota universe.

Installation: 
1. Clone repository
2. Open project in Vs Code 
3. Run the Spring Boot application
4. The API will run on: http://localhost:8080/api/characters/ 

Api Endpoints:

GET /characters
GET /characters/{id}
POST /characters
PUT /characters/{id}
DELETE /characters/{id}
GET /characters/category/{category}  
GET /characters/search?name=substring
